# EventureAI App Builder

A dual-interface tool for building modern applications.

## Packages

- `packages/cli`: A TypeScript-based Command Line Interface for scaffolding and generation.
- `packages/web`: A Next.js Web UI for visual app construction.

## Getting Started

### Prerequisites
- Node.js (v18+)
- npm

### Installation
Run from the `builder` directory:
```bash
npm install
```

### Running the CLI
```bash
# Build and run the CLI
npm run cli
```

### Running the Web UI
```bash
# Start the Next.js dev server
npm run web
```

## Features
- [x] Monorepo scaffold (npm workspaces)
- [x] CLI with project initialization logic
- [x] Next.js Web UI with Tailwind CSS
- [ ] Shared Core logic
- [ ] Visual Drag & Drop
- [ ] Code Generation Templates
